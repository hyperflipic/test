# AI工具导航

一个简洁的AI工具导航网站，帮助用户快速找到并使用各种AI工具。

## 功能特点

- 分类展示：对话、图像、音乐、视频、开发、PPT等多个分类
- 双语支持：中文和英文
- 深色/浅色主题切换
- 响应式设计：完美适配移动端和桌面端
- 简洁美观的卡片式布局

## 本地运行

1. 克隆仓库
```bash
git clone https://github.com/[你的用户名]/ai-tools-nav.git
```

2. 在浏览器中打开 `index.html` 文件
